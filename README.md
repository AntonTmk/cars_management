# Cars managment
You can use this program to search for cars from the cars.yml file with the parameters you want
## Available filters:
* make
* model
* year_from
* year_to
* price_from
* price_to
Also you can chose sort option
## Avaliable sort options:
* date_aded asc
* date_aded desc
* price asc
* price desc
## How to use this app
* Install Ruby 3.*
* clone the repository
* Run start.rb
